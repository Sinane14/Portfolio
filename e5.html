Laclick Project Forum

Présentation

Ce Forum permet aux utilisateurs de se retrouver afin de découvrir de nouveaux sujets; de partager leurs nouvelles passions et de donner leur avis. Le Forum vise les étudiants en BTS SIO de La Click pour leur permettre de présenter leurs projets.

Procédure d'accès et d'utilisation

Laclick Project Forum est une application web essentiellement en php et qui fonctionne avec une base de donnée. Veuilliez télécharger la base de donnée et de l'importer sur XAMPP. Si ce n'est pas déjà fait, vous devez installer l'application XAMPP. Allez sur le site xampp.

Projet accessible sur github au lieu suivant : Lien Github

Le Super Taquin

Présentation

Le Taquin est un jeu ayant pour objectif de former une image à l'aide de case le plus rapidement possible.

Procédure d'accès et d'utilisation

Le Super Taquin est une application executable en C# et qui fonctionne avec la base de donnée du Forum. Veuilliez télécharger la base de donnée et de l'importer sur XAMPP. Si ce n'est pas déjà fait, vous devez installer l'application XAMPP. Allez sur le site xampp.

Projet accessible sur google drive au lien suivant: Lien Google Drive
