Laclick Project Forum

Présentation

Ce Forum permet aux utilisateurs de se retrouver afin de découvrir de nouveaux sujets; de partager leurs nouvelles passions et de donner leur avis. Le Forum vise les étudiants en BTS SIO de La Click pour leur permettre de présenter leurs projets.

Procédure d'accès et d'utilisation

Laclick Project Forum est une application web en php qui fonctionne avec une base de donnée. Si ce n'est pas déjà fait, vous devez installer l'application XAMPP pour accéder à PhpMyAdmin. Pour cela, je vous invite à suivre le tutoriel sur youtube sur l'installation de XAMPP: Tuto (Jusqu'à 2min23). Importer les fichiers forum du github et les décompresser dans le fichier htdocs de XAMPP: xampp/htdocs. Dans l'application XAMPP, lancez Apache et Mysql. Appuyez sur le bouton Admin de MySQL. Veuilliez importer la base de donnée accessible au lien github du projet sur PhpMyAdmin. Activer Apache et MySQL sur XAMPP. Enfin, allez sur le navigateur et tapper localhost.

Le Super Taquin

Présentation

Le Taquin est un jeu ayant pour objectif de former une image à l'aide de case le plus rapidement possible.

Procédure d'accès et d'utilisation

Installation

Le Super Taquin est une application executable en C# et qui fonctionne avec la base de donnée du Forum. Si ce n'est pas déjà fait, vous devez installer l'application XAMPP pour accéder à PhpMyAdmin. Pour cela, je vous invite à suivre le tutoriel sur youtube sur l'installation de XAMPP: Tuto (Jusqu'à 2min23). Importer et décompresser les fichiers Taquin du github. Télécharger Visual Studio 2022. Pour cela, je vous invite à suivre le tutoriel sur youtube pour l'installation: Tuto Ouvrir le jeu Taquin.sln sur Visual Studio 2022. Dans l'application XAMPP, lancez Apache et Mysql. Veuilliez importer la base de donnée accessible au lien github du projet sur PhpMyAdmin. Enfin, démarrer le code.

Connexion

Pour pouvoir jouer, l'application vous demande de vous connecter. Le jeu est accessible uniquement aux utilisateurs du Forum. Vous devez alors être un utilisateur du Forum. Pour cela, vous devez créer un compte utilisateur dans le Forum: S'inscrire Quand l'inscription est faite, vous vous connectez avec vos identifiants dans le jeu et vous pouvez jouer.
