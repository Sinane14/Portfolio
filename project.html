Site web pour les entreprises

En première année du BTS, j'ai effectué un stage en distanciel avec une entreprise où j'étais chargé de proposer un site web pour celle-ci. Accédez directement au fichier du site sur github à extraire dans les fichiers xampp: Github Si ce n'est pas déjà fait, vous devez installer l'application XAMPP pour accéder à PhpMyAdmin. Pour cela, je vous invite à suivre le tutoriel sur YouTube sur l'installation de XAMPP: Tuto (Jusqu'à 2min23). Le chemin à suivre pour le dépot des fichiers est le suivant: C:\xampp\htdocs. Dans l'application XAMPP, lancez Apache et Mercury. Pour accéder au site, tappez sur la barre de navigation: localhost.
